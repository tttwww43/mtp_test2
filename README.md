# Korean Medical Training Program (KMTP) Website

## 🏥 프로젝트 개요

외국인 의사들을 위한 한국 의료 연수 프로그램 지원 웹사이트입니다. 의료 기술이 세계적으로 인정받는 한국에서의 연수 기회를 제공하는 종합적인 플랫폼입니다.

## ✨ 주요 기능

### 🌐 다국어 지원
- 한국어/영어 전환 기능
- 모든 콘텐츠의 완전한 번역 지원

### 📝 온라인 지원 시스템
- 외국인 의사를 위한 지원서 작성
- 파일 첨부 기능 (이력서, 자격증)
- 실시간 폼 유효성 검사
- 지원 상태 추적

### 📊 데이터 관리
- 지원자 정보 데이터베이스
- RESTful API를 통한 데이터 관리
- 자동 생성되는 시스템 필드 (ID, 생성일, 수정일 등)

### 📱 반응형 디자인
- 데스크톱, 태블릿, 모바일 최적화
- **초고급 UI/UX 디자인**
- 부드러운 애니메이션 효과

## 🎨 초고급 UI/UX 특징

### 🌟 고급스러운 디자인 요소
- **유리모피즘(Glassmorphism)**: 반투명한 배경과 블러 효과
- **프리미엄 그라디언트**: 귀금속 느낌의 고급스러운 색상 팔레트
- **3D 입체감**: 레이어드 섀도우와 호버 효과
- **마이크로 인터랙션**: 섬세한 애니메이션과 전환 효과

### 🎯 고급 애니메이션
- **페이드인/업/다운/레프트/라이트**: 다양한 방향의 부드러운 등장 효과
- **펄스 애니메이션**: 생동감 있는 아이콘 효과
- **바운스 애니메이션**: 활기찬 느낌의 점프 효과
- **셔머 효과**: 프로그래스 바의 고급스러운 애니메이션

### 💎 프리미엄 컴포넌트
- **고급 버튼**: 그라디언트 배경과 글로우 효과
- **입체 카드**: 호버 시 들어올림과 확대 효과
- **세련된 폼**: 유리모피즘 디자인의 입력 필드
- **고급 파일 업로드**: 드래그 앤 드롭 인터페이스

### 🌈 컬러 시스템
- **프라이머리 그라디언트**: 블루-퍼플-핑크의 고급스러운 조합
- **세컨더리 그라디언트**: 그린 톤의 자연스러운 변화
- **액센트 그라디언트**: 핑크-레드의 활기찬 조합
- **다크 테마**: 깊이 있는 레이어드 배경

## 🚀 현재 구현된 기능

### 메인 페이지 (index.html) - **초고급 디자인**
- **고급 히어로 섹션**: 배경 애니메이션과 페이드인 효과
- **프리미엄 CTA 버튼**: 글로우 효과와 호버 애니메이션
- **입체 특징 카드**: 호버 시 들어올림과 확대 효과
- **통계 섹션**: 고급스러운 숫자 표시
- **푸터**: 연락처 정보와 함께 고급스러운 디자인

### 지원 페이지 (application.html) - **고급 폼 시스템**
- **유리모피즘 폼**: 반투명한 배경과 블러 효과
- **실시간 유효성 검사**: 시각적 피드백과 에러 메시지
- **고급 파일 업로드**: 드래그 앤 드롭 인터페이스
- **프로그레스 표시**: 셔머 애니메이션 효과
- **지원 자격 및 필요 서류**: 고급스러운 카드 레이아웃

### 프로그램 소개 (program.html) - **입체 카드 디자인**
- **프리미엄 프로그램 카드**: 호버 시 들어올림과 글로우 효과
- **연수 분야 상세**: 6개 분야의 고급스러운 카드 표시
- **타임라인**: 시각적 흐름의 고급 디자인
- **연수 혜택**: 아이콘과 함께하는 고급스러운 레이아웃
- **FAQ 섹션**: 인터랙티브한 접기/펼치기 기능

### 연락처 페이지 (contact.html)
- **연락처 정보 카드**: 유리모피즘 디자인
- **문의 폼**: 고급스러운 입력 필드
- **자주 묻는 질문**: 인터랙티브한 아코디언
- **소셜 미디어**: 고급스러운 아이콘 링크

## 📋 데이터 스키마

### applicants 테이블
- `id`: 고유 식별자
- `name`: 의사 이름
- `email`: 이메일 주소
- `nationality`: 국적
- `medical_license`: 의사 면허 번호
- `specialization`: 전문 분야
- `hospital`: 소속 병원
- `experience_years`: 경력 년수
- `preferred_program`: 희망 연수 프로그램
- `duration`: 연수 기간
- `korean_level`: 한국어 수준
- `status`: 지원 상태 (접수, 심사중, 승인, 거절)
- `message`: 지원 동기 메시지
- `application_date`: 지원일

## 🔧 API 엔드포인트

### 지원자 데이터
```
GET    /tables/applicants          # 지원자 목록 조회
POST   /tables/applicants          # 새 지원자 등록
GET    /tables/applicants/{id}       # 특정 지원자 조회
PUT    /tables/applicants/{id}       # 지원자 정보 수정
DELETE /tables/applicants/{id}     # 지원자 정보 삭제
```

## 📱 반응형 지원

- **데스크톱**: 1200px 이상 (고급스러운 레이아웃)
- **태블릿**: 768px - 1200px (최적화된 카드 레이아웃)
- **모바일**: 768px 이하 (터치 친화적 인터페이스)

## 🚀 시작하기

1. **파일 다운로드**: 모든 HTML, CSS, JS 파일을 다운로드
2. **브라우저에서 실행**: index.html 파일을 브라우저로 열기
3. **지원서 작성**: application.html에서 고급스러운 지원서 작성
4. **데이터 확인**: 브라우저 개발자 도구에서 API 응답 확인
